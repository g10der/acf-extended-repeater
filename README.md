# G10DER Repeater Fields for Advanced Custom Fields

A lightweight repeater field extension for Advanced Custom Fields (ACF) Free.

## Features

- Repeater Field for ACF Free
- Drag & Drop Row Sorting
- Row Collapse / Expand
- Import & Export Support
- WYSIWYG Support
- Date Picker Support
- Date Time Picker Support
- Image and File Fields

## Installation

1. Install Advanced Custom Fields Free.
2. Install and activate G10DER Repeater Fields.
3. Create a field group.
4. Add the Repeater field.
5. Add sub-fields and save.
